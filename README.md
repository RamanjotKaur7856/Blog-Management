# Blog Management System

## 🚀 Overview
A full-stack blog application that allows users to create, edit, and delete posts.

## 🛠 Tech Stack
- Node.js  
- Express  
- MongoDB  
- EJS  

## ✨ Features
- Create, edit, and delete blog posts  
- RESTful routing  
- Dynamic content rendering using EJS  

## 📌 What I Learned
- CRUD operations in web applications  
- Backend routing and API design  
- Database integration with MongoDB
